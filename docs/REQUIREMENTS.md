# sunyuDX-flow 要件定義（確定版）

## 目的
Excel工程表運用をそのままWeb化し、
見積 → 受注 → 工事台帳 → 工程表 → 原価 → 差異 を一気通関で管理する。

## 絶対ルール
- 工程表が主役
- 手入力スタート禁止（Excel/PDF起点）
- 色は2色のみ（夜間=黄色 / 人材不足=赤）
- 請負外注は人数管理しない（数量のみ）

## ユーザー
- 現場（スマホ）：日報・確認
- 工事担当（PC）：工程・配置・外注
- 事務/経営（PC）：見積・原価・集計

## 原価分類
- 労務 / 外注 / 材料 / 機械 / 経費

## 工程表ルール
- 現場×日(週) / 現場×週(月)
- 社員は名前表示
- 外注はラベル表示
- 3名超は +N 省略

## 完成条件
- 工程表がExcelと同じ感覚で操作できる
- 赤と黄色の意味を誰でも即理解できる
